# 🎯 Number Guessing Game (Python)

## 📌 Overview
This is a simple command-line **Number Guessing Game** written in Python.  
The program generates a random number between **1 and 100**, and the player has **10 chances** to guess it correctly.

---

## 🚀 Features
- 🎲 Random number generation  
- 👤 Personalized user interaction (asks for your name)  
- 🔁 Multiple guesses with hints  
- 📉 Limited attempts (10 chances)  
- 💬 Friendly prompts and responses  
- 🏁 Win/Lose result display  

---

## 🧠 How the Game Works
1. The program asks for your name  
2. It asks if you want to play  
3. If you agree:
   - A random number (1–100) is generated  
   - You get **10 chances** to guess it  
4. After each guess:
   - You get a hint (Too high / Too low)  
5. Game ends when:
   - You guess correctly ✅  
   - OR run out of attempts ❌  


# Run the program
python game.py
